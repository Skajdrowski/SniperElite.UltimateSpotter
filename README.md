# Ultimate Spotter for Sniper Elite
(Originally known as "Spotter") An ASI script to moderate online lobbies in Sniper Elite 2005 (**Steam version only**)

## How to install
Extract [archive content](https://github.com/Skajdrowski/SniperElite.UltimateSpotter/releases/latest) (2 files) into game directory (eg. C:\Program Files\Steam\steamapps\common\SniperElite) and run the game.

If you have already installed Widescreen fix mod, you only have to drop .asi file into either the game files or scripts folder.

## Usage
Press INSERT while being in Main Menu, to open up Ultimate Spotter's interface.
